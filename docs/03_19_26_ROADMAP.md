# Project Roadmap

## Phase 1 — Foundation (Current)

- Build parser.
- Build positional AST.
- Implement basic evaluator in Python.
- Add JSON export for AST.

## Phase 2 — Runtime Expansion

- Add functions, classes, and scopes.
- Implement basic standard library.
- Improve error handling.

## Phase 3 — Tooling & Ecosystem

- Expand BInspected for runtime introspection.
- Split FigMan into:
  - FigConfig (configuration)
  - FigAST (AST utilities)
- Finish and publish DesignMe as a portfolio piece.

## Phase 4 — Performance & Portability

- Design bytecode format.
- Build a simple VM.
- Explore C or Rust runtime.

## Phase 5 — Bootstrapping

- Rewrite parts of the interpreter in Dub itself.
- Move toward self-hosting.
